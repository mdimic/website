Projects

J.A.R.V.I.S.

JARVIS is an intuitive voice controlled smart motorcycle helmet.

Developed for WearHacks Waterloo. Won first place and community favourite prizes. Also got featured on the DevPost front page.

Technologies used: Node.js, AWS, Amazon Alexa, Raspberry Pi 2, Firebase, Twilio.

City Explorer

City Explorer is an itinerary planner that shows you the best attractions in each city and allows you to plan the most optimal route.

Developed for McHacks. Won best use of TripAdvisor API prize.

Technologies used: AngularJS, TripAdvisor, HTML, CSS.

Music Player

A music player developed in C for the Altera DE2 board.

Implemented various features such as play/pause, skip, fast forward, reverse, etc.

Developed for ECE 224 - Embedded Microprocessor Systems. Received a grade of 100% on the project.

Technologies used: C

Alexa Keep

Alexa Keep is a skill for Amazon Alexa that allows users to store any information they might want to know in the future.

Developed for the Connected Lab Amazon Alexa Hackathon. Placed in the top 5 teams.

Technologies used: Amazon Echo, node.js, AWS, Amazon Alexa, DynamoDB.
